## 1. Website Address
**jayadhaba.com** — customers scan QR codes to open this

## 2. Admin Dashboard
**jayadhaba.com/admin** — manage everything here

## 3. How to Make an Item Unavailable ("86 It")
1. Open **jayadhaba.com/admin**
2. Click **Menu Manager** (left sidebar)
3. Find the item
4. Click the **Available** toggle → turns OFF
5. Done — customers see it greyed out immediately

## 4. How to Update Order Status
1. Open **jayadhaba.com/admin**
2. Click **Orders Manager**
3. Click the order number
4. Choose new status:
   - **Preparing** = kitchen started cooking
   - **Ready** = food is at the counter
   - **Served** = delivered to table / handed over
5. Customer phone shows the update automatically

## 5. How to Generate a New QR Code
1. Open **jayadhaba.com/admin**
2. Click **Settings Manager**
3. Click **Generate QR Codes**
4. Download the PNG file
5. Print and place on the table

## 6. Payment Not Showing?
- Check Razorpay dashboard: **dashboard.razorpay.com**
- If payment is there but order is not confirmed → call support
- Do NOT give food until payment shows "success"

## 7. AI Chatbot Not Working?
- It needs internet — check WiFi
- If still not working → customers can still call the number below

## 9. Daily Checklist
- [ ] Check **Orders Manager** for new orders
- [ ] Check **Menu Manager** — mark unavailable items

## 10. Emergency Restart
If the tablet / screen is frozen:
1. Hold power button 10 seconds
2. Turn back on
3. Open browser → jayadhaba.com/admin
4. Log in again
